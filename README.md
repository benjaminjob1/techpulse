# TechPulse — AI-Powered Tech Stock Dashboard

A real-time tech stock dashboard with AI-powered analysis and price forecasting. Built with React, Express, and Tailwind CSS.

## Features

- **Live Market Data** — Real-time stock quotes, market movers (gainers/losers/most active), and market sentiment
- **15 Tech Stocks** — NVDA, AAPL, GOOGL, MSFT, AMZN, AVGO, META, TSLA, ORCL, NFLX, PLTR, AMD, CRM, INTC, ADBE
- **AI Stock Analysis** — Buy/Sell/Hold signals with confidence levels, bull/bear cases, and key factors
- **Price Forecasting** — AI-generated price predictions across 5 time horizons (1W, 1M, 3M, 6M, 1Y) with range estimates
- **Interactive Charts** — 3-month price history, market cap comparison, sparklines
- **Analyst Consensus** — Real analyst ratings, price targets, and rating breakdowns
- **Dark/Light Mode** — Full theme support
- **Mobile Responsive** — Works on all screen sizes
- **Auto-Refresh** — Data refreshes every 5 minutes

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, shadcn/ui, Recharts, TanStack Query
- **Backend**: Express.js, Node.js
- **AI**: Anthropic Claude (for analysis and forecasting)
- **Data**: Live market data via finance API connector
